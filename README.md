# Extension WanaPdfViewer
Volodymyr Dvernytskyi nous offre une extension nommée 'PDF Viewer' (Open Source).

Outre la visualisation d’un pdf incorporé à Business Central (désormais standard), l’extension elle permet d’ajouter des documents joints à des fonctions qui n’en disposent pas dans la version standard. 

L'extension WanaPdf l'applique aux fonctions ci-après et l'enrichit de quelques fonctionnalités.

![WanaPdfViewer](images/WanaPdfViewer.png)

### Sommaire

- [Clients](#clients)
- [Fournisseurs](#fournisseurs)


## Clients

Des PDF peuvent être joints aux **Comptes bancaires clients** ainsi qu'aux **Mandats de prélèvement**.

## Fournisseurs
Des PDF peuvent être joints aux **Comptes bancaires fournisseurs**.

Associée à l'extension [WanApprove](https://github.com/Wanamics/WanApprove?#extension-wanapprove) cette extension facilitera l'approbation.
